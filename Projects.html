  • Torn.com importing profit calculator


    Description: Torn.com is a MMORPG text-base game, there is a function in the game let players importing flowers and plushies from another countries for profit. There are many factorials that affect the profit. This project is made to calculate the profit just in clicks.


      3 functions:

    1. 1. Fetch API from Torn.com to get average market price of 21 different items which change continuously
    2. 2. Calculate the profit and profit per minute for each item imported.
    3. 3. Sorting function for the table allow user to sort any column.

    How to use: Choose 1 out of 4 tickets (Standard, Airstrip, Private, Business). Each player may have different ticket.
